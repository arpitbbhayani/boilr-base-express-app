This is a boilr template for a basic nodeJS app.
This works on delimiter {{@@@@}} built on my custom fork of boilr.

## Setting up Development Environment

```
cd {{@@Name@@}}/{{@@Name@@}}
```

```
npm install
```

In one Tab
```
npm run build-dev
```

In Second Tab
```
npm run run-dev
```

In Third Tab
```
npm run copy-statics-dev
```

Before committing

```
npm run lint
```
